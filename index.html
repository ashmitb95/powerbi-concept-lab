Power BI Concept Lab

An interactive, free web app that teaches the invisible parts of Power BI — filter context, relationships, measures, and DAX — on data you choose, then hands you off to Power BI Desktop to build for real. For someone who can already make simple dashboards and wants to reach intermediate/advanced.

It's an interactive app — please enable JavaScript to use it.

AI assistants & crawlers

A machine-readable guide to this app — its tabs, datasets, concepts, recommended learning path, and how to coach a learner through it (including how to set up a Claude Project) — is published at /llms.txt (also /agents.txt). Fetch that to help a user get started.

What's inside

  • Pick a dataset (Northwind, Coffee Shop, SaaS Revenue, or your own CSV); every concept tunes to its columns.
  • Interactive trainers: filter context, measure vs calculated column, relationships, ranking (RANKX), time intelligence (YoY/MoM/YTD).
  • A searchable reference of concepts, patterns and ~50 DAX functions, each linking to the exact YouTube timestamp.
  • An "Ask Claude to guide me" prompt and a built-in Claude tutor setup.

The learning loop: learn a concept here → open the same data in Power BI Desktop → ask your Claude tutor when stuck.
